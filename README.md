## Overview

VoxClaw enables AI agents (like OpenClaw) to make real phone calls and have autonomous conversations to complete business tasks.

## Use Cases

- Call businesses to get information
- Make appointments
- Verify data with customer service
- Navigate phone trees (IVR)
- Follow up on pending requests

## Tech Stack

- **Telephony:** Twilio
- **Speech-to-Text:** Whisper
- **Text-to-Speech:** ElevenLabs / OpenAI
- **LLM:** Claude / GPT-4
- **Runtime:** Node.js
